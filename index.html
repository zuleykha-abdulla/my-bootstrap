Bootstrap nədir

Column 1

Column 2

Bootstrap-da formalar, düymələr, cədvəllər, naviqasiya, modallar, şəkil karuselləri və bir çox başqa elementlər üçün HTML və CSS əsaslı dizayn şablonları, həmçinin əlavə JavaScript plaginləri daxildir.

Column 3

Bootstrap vasitəsi ilə siz asanlıqla mobil responsive dizaynlar yaratmaq imkanı verir

Bootstrap Grid Sistemi

  • Bootstrap-ın grid sistemi flexbox ilə qurulub və səhifədə 12-yə qədər sütun hazırlamağa imkan verir.Grid-lər həmişə row – klasının içində olur
  • Bütün 12 sütunu ayrı-ayrılıqda istifadə etmək istəmirsinizsə, daha geniş sütunlar yaratmaqla sütunları birlikdə qruplaşdıra bilərsiniz:
  • Grid sistemi responsivdir və sütunlar ekran ölçüsündən asılı olaraq avtomatik olaraq yenidən formalaşır.
  • Sütünların cəminin 12 və ya daha az əlavə olunduğundan əmin olun (bütün 12 mövcud sütundan istifadə etməyiniz tələb olunmur).

Grid Classes 5 dənədir

  • .col- (extra small devices - screen width less than 576px)
  • .col-sm- (small devices - screen width equal to or greater than 576px)
  • .col-lg- (large devices - screen width equal to or greater than 992px)
  • .col-xl- (xlarge devices - screen width equal to or greater than 1200px)
